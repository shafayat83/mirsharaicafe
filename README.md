# Mirsarai Cafe ☕

A modern, responsive, and highly animated website for **Mirsarai Cafe**, located in Mirsharai, Bangladesh. This project focuses on providing a premium user experience with smooth scroll-based animations, full accessibility, and a mobile-first design.

**Live Demo:** [https://mirsharaicafe2.netlify.app/](https://mirsharaicafe2.netlify.app/)

---

## ✨ Features

- **🌊 Scroll-Based Animations:** Powered by Framer Motion, sections gracefully fade and slide into view as you explore.
- **📱 Fully Responsive:** Optimized for all devices—from small mobile screens to large 4K monitors.
- **♿ Accessibility First:** Includes ARIA attributes, skip-to-content links, and full keyboard navigation support.
- **🚀 High Performance:** Built with Vite for lightning-fast development and optimized production builds.
- **🎨 Modern UI/UX:** A warm, cafe-inspired color palette with elegant typography.
- **📍 Interactive Elements:**
  - Custom mobile sidebar navigation.
  - Floating "Scroll to Top" button.
  - Interactive Menu with hover effects.
  - Contact form with validation.
  - Embedded Google Maps for easy location finding.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/mirsharai-cafe.git
   cd mirsharai-cafe
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```
   The app will be available at `http://localhost:3000`.

### Building for Production

To create an optimized production build:
```bash
npm run build
```
The output will be generated in the `dist/` directory.

## 🌐 Deployment

### Netlify (Current)
The project is configured for easy deployment on Netlify. Simply connect your GitHub repository and set the following:
- **Build Command:** `npm run build`
- **Publish Directory:** `dist`

### Vercel
A `vercel.json` file is included for SPA routing.
- **Framework Preset:** Vite
- **Build Command:** `npm run build`
- **Output Directory:** `dist`
